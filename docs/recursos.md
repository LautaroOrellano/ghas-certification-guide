## Best Practices Checklists

### Initial Setup Checklist

```
□ Enable GHAS for organization
□ Configure default settings for new repos
□ Enable Dependabot alerts
□ Enable Dependabot security updates
□ Enable secret scanning
□ Enable push protection
□ Configure CodeQL default setup
□ Setup branch protection rules
□ Configure notifications
□ Create security team
□ Assign security managers
□ Document security policies
□ Train development teams
```

### Daily Operations Checklist

```
□ Review new critical/high alerts
□ Triage Dependabot PRs
□ Monitor CodeQL workflow runs
□ Check secret scanning alerts
□ Review push protection bypasses
□ Support developers with security questions
```

### Weekly Review Checklist

```
□ Security metrics dashboard review
□ Audit dismissed alerts
□ Review SLAs (alerts > 7 days old)
□ Update custom patterns
□ Review workflow performance
□ Team security standup
```

### Monthly Tasks

```
□ Generate security report
□ Review all accepted risks
□ Update security policies
□ Review tool configuration
□ Audit access permissions
□ Review budget and usage
□ Security training sessions
□ Update documentation
```

### Severity Levels

```
Critical: Fix immediately (< 24h)
High:     Fix within sprint (< 7 days)
Medium:   Fix within month (< 30 days)
Low:      Technical debt (< 90 days)
```

### CVSS Score Ranges

```
0.0:       None
0.1-3.9:   Low
4.0-6.9:   Medium
7.0-8.9:   High
9.0-10.0:  Critical
```

### Common CWEs

```
CWE-79:  XSS
CWE-89:  SQL Injection
CWE-22:  Path Traversal
CWE-78:  OS Command Injection
CWE-502: Unsafe Deserialization
CWE-798: Hardcoded Credentials
```
